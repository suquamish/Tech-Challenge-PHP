# Tech-Challenge-PHP

Brushing up on PHP

Build a RESTFUL service using PHP and Laravel to manage "users":

- create a member
- get all members
- delete a member
- update a member

Build a front end using Vue.js:

- add a new member
- search for member by username
- display a member's data
- update a member's email and/or name

Member Model:
- username
- email
- name
