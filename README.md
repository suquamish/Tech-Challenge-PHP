# Tech-Challenge-PHP

Brushing up on PHP

Build a RESTFUL service using PHP and Laravel to manage "users":

- create a user
- get all users
- delete a user
- update a user

Build a front end using Vue.js:

- add a new user
- search for user by username
- display a user
- update a user's email and/or name

User Model:
- username
- email
- name
